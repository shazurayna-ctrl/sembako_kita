# 📖 Panduan Pengguna — SembakoKita.Pro

**Ekosistem Krisis Pangan — Offline-first, Mesh Network, AI Agentic**

---

## 📌 Apa Itu SembakoKita.Pro?

SembakoKita.Pro adalah aplikasi yang membantu **pedagang kecil, warung sembako, dan komunitas** dalam menghadapi krisis pangan. Aplikasi ini bisa jalan **tanpa internet**, memakai **Bluetooth Mesh** untuk komunikasi antar HP, dan punya **AI Assistant** yang siap membantu.

---

## 🚀 Memulai

### 1. Install Aplikasi

- **Android**: Download file `.apk` dari link yang diberikan, lalu install.
- **Windows**: Download file `.exe` dan jalankan.

### 2. Registrasi / Login

- Buka aplikasi, pilih **"Daftar"**.
- Isi nama, nomor HP, dan NIK.
- Verifikasi via SMS/WhatsApp.

### 3. Mode Aplikasi

| Mode | Warna | Kapan Dipakai |
|------|-------|---------------|
| **Normal** | Hijau | HP bagus, baterai penuh |
| **Krisis** | Merah | HP menengah, baterai sedang |
| **Survival** | Hitam-putih | HP tua, baterai rendah |

Aplikasi akan otomatis pilih mode yang sesuai.

## 🆘 Darurat

### Tombol SOS
- Tekan tombol merah di pojok kanan atas.
- Isi pesan darurat.
- Sinyal akan dikirim ke semua node dalam radius 2.4 km.

### Laporan Darurat
- Pilih **"Laporan Baru"**.
- Pilih prioritas **"Darurat"**.
- Isi judul & deskripsi.
- Laporan akan langsung masuk ke checklist posko.

## ❓ FAQ

**Q: Apakah aplikasi bisa jalan tanpa internet?**
A: Ya! Semua fitur jalan offline. Sinkronisasi otomatis saat online.

**Q: Berapa ukuran APK?**
A: Sekitar 15-30 MB (tergantung fitur yang diinstall).

**Q: Apakah bisa dipakai di HP jadul?**
A: Minimal Android 5.0 (API 21), RAM 1GB.

**Q: Bagaimana cara bantu pengembangan?**
A: Fork repo di GitHub, buat pull request, atau donasi.
